---
title: Open-source license
description: Decision doc for how we license our work
---

As part of our principle to be [open/public by default](/innovation-engineering/onboarding/2-principles#openpublic-by-default-closedprivate-when-necessary), we license our software under the **MIT License**, with copyright held by the New Jersey Innovation Authority. Every repository should include a `LICENSE` file at the root with the copyright line: `Copyright (c) <year> New Jersey Innovation Authority`.

For the full rationale behind this decision, see the [Open Source License decision doc](https://docs.google.com/document/d/1d80pbXd1QMrkPMifWSzlJGkboq8uhWkPTh6CJIl5aw4/edit?usp=sharing).
