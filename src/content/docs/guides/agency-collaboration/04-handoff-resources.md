---

The Doula Medicaid project has gone through onboarding agency stakeholders to make small text changes, and to steward renovatebot pull requests that are not automatically merged.

The onboarding sessions were structured as:

1. The agency IT team handled installation of VSCode, git, and other tools, and cloned the repository via VSCode
1. Global search in VSCode to change text
1. Running tests and checks, and setting up eslint and prettier in VSCode
1. Pulling main, creating a new branch, and using GitHub to make a pull request
1. Doing a ticket
1. Evaluating and merging renovatebot pull requests

Onboarding session materials can be found in [this folder](https://sonj.sharepoint.com/:u:/s/OIT-Innovation/DAP/IQC6xwP1TcjRRJQ0C71DABlYAXvRP0jCO9H5BLTXlzo1z2M?e=YuqKCK).
